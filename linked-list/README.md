# 📁 Linked List

Includes singly and doubly linked list problems.

## Topics Covered:
- Node creation and traversal
- Insertions & deletions
- Reversing a list
- Detecting cycles
- Finding the middle node

## Example Problems:
- Reverse Linked List
- Detect Cycle in Linked List
- Merge Two Sorted Lists
- Remove N-th Node From End of List
